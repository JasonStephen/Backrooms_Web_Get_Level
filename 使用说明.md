# Backrooms 爬虫使用说明

## 启动

在项目根目录：

```powershell
python .\scripts\backrooms_crawler.py --mode test
```

在 `scripts` 目录：

```powershell
python .\backrooms_crawler.py --mode test
```

## 模式

测试模式：只读 `samples/`

```powershell
python .\backrooms_crawler.py --mode test
```

随机模式：随机抽取若干层级

```powershell
python .\backrooms_crawler.py --mode random --count 10
python .\backrooms_crawler.py --mode random --count 10 --seed 1
```

随机非 404 模式：直到凑够指定数量的非 404 页面

```powershell
python .\backrooms_crawler.py --mode random-non404 --count 10
python .\backrooms_crawler.py --mode random-non404 --count 10 --seed 1
```

指定层级模式：抓取一个或多个指定层级

```powershell
python .\backrooms_crawler.py --mode specific --levels level-187
python .\backrooms_crawler.py --mode specific --levels level-138 level-176 level-187
```

完整模式：遍历全部目标，默认关闭

```powershell
python .\backrooms_crawler.py --mode complete
```

调试输出示例：

```powershell
python .\backrooms_crawler.py --mode complete --debug
```

## 输出

结果都在 `outputs/`：

- `outputs/<mode>_results.txt`
- `outputs/html/*.body.html`
- `outputs/md/*.body.md`

## 配置

配置文件：[crawler_config.json](E:/Project/Backrooms/Backrooms_Web_Get_Level/crawler_config.json)

常用项：

- `whitelist_extra_paths`：额外白名单层级
- `preferred_paths`：特殊映射，例如 `level-0 -> latest:level-0`
- `worker_count`：并发抓取线程数，默认 `4`
- `complete_mode_enabled`：是否开启完整模式
- `output_version`：输出版本号，用于跳过同版本、覆盖旧版本
